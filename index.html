Encryptionsuite

A command-line tool for encrypting and decrypting files and folders. It secures data through obfuscate encryption using personal secret keys and allows for multiple encryption layers, enabling content to be encrypted up to three times. Users are advised to handle encryption keys carefully to prevent data loss.

AFReader

A command-line tool that animates and displays file content, allowing for easy reading from the terminal. Users can install the tool via pip and utilize it to read files with animated effects directly in the command line.

ThreatHunter

fconverter

A Python-based command-line interface (CLI) utility designed for converting documents between various formats. It supports conversions such as Word to PDF, PDF to Word, Word to PPT, Word to TXT, PDF to TXT, PPT to Word, TXT to Word, TXT to MP3 (using Google Text-to-Speech), image enhancement to grayscale, and XLSX to CSV. The tool is installable via pip and offers a range of options for different file conversions.

CryptoSharePKI

This project implements a Public Key Infrastructure (PKI) system. It includes scripts and tools for generating and managing certificates, setting up root and intermediate certificate authorities, and distributing certificates. The repository contains various Python scripts and configuration files essential for establishing a PKI system.

Agrimax

A comprehensive Crop Recommendation and Field Management System aimed at empowering farmers with data-driven decisions to enhance crop yield and farm management. Agrimax integrates soil health analysis, weather forecasting, and crop recommendation features to assist users in making informed agricultural choices. Key functionalities include suggesting optimal crops based on soil nutrients and environmental factors, evaluating soil health with actionable insights, and providing weather forecasts to inform farming activities.
